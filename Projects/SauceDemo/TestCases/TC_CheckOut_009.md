TC-009

Title: Incorrect First Name, Second Name, Zip/Postal Code, instead of string and numbers put more than 100 "a"

Preconditions: User is on Products page

Test Steps:

  1. Click product: Sauce Labs Bike Light
  2. Click Add to cart
  3. Click icon: Cart
  4. Click Checkout
  5. Enter First Name: 100 and more "a"
  6. Enter Second Name: 100 and more "b"
  7. Enter Zip/Postal Code: 100 and more "c"
  8. Click Continue

  
Expected Result: Error message is displayed.
