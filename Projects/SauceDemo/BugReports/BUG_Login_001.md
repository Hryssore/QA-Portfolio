Title:
Error message overlaps Login button when invalid credentials are entered

Severity:
Minor

Priority:
Low

Environment:
Windows 11
Google Chrome
SauceDemo Login Page

Preconditions:
User is on the login page.

Steps to Reproduce:
Open https://www.saucedemo.com
Enter valid username standard_user
