Title:
Error message overlaps Login button when invalid credentials are entered

Severity:
Minor

Priority:
Low

Environment:
Windows 11
Google Chrome
SauceDemo Login Page

Preconditions:
User is on the login page.

Steps to Reproduce:

  1. Open https://www.saucedemo.com. 
  2. Enter valid username standard_user. 
  3. Enter invalid password. 
  4. Click Login. 

Actual Result:
The error message extends beyond its container and overlaps the Login button, reducing readability and negatively affecting the user interface.

Expected Result:
The error message should remain fully inside the designated container without overlapping other UI elements.
